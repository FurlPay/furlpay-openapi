# Contributing

Thanks for helping build the Furlpay ecosystem!

- **Bugs / features** — open an issue with a minimal reproduction.
- **Pull requests** — keep them focused; add tests where the package has them.
- **Questions** — use GitHub Discussions on this repo or the org.

This package is developed in the internal Furlpay monorepo and mirrored here;
maintainers sync accepted PRs upstream. By contributing you agree your work is
licensed under the MIT license of this repository.
